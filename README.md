# BrewSpace Coffe Shop ☕

A modern and responsive Coffee Shop landing page built using HTML and Tailwind CSS.

## 🚀 Features

- Responsive Design
- Smooth Scrolling Navigation
- AOS Scroll Animation
- Modern UI Layout
- Mobile Friendly

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- AOS (Animate On Scroll)
